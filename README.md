# Proyecto Final Sprint 7 - Análisis de Clientes ConnectaTel

## Objetivo del proyecto

Realizar un análisis exploratorio de datos (EDA) para comprender el comportamiento de los clientes de ConnectaTel. El proyecto busca identificar patrones de uso, segmentar a los clientes por edad y nivel de consumo, detectar valores atípicos y generar recomendaciones que apoyen la toma de decisiones del negocio.

## Datasets utilizados

El análisis se realizó utilizando los siguientes archivos:

* **plans.csv:** Información de los planes ofrecidos por ConnectaTel.
* **users_latam.csv:** Datos demográficos de los clientes.
* **usage.csv:** Información sobre el uso de llamadas y mensajes de cada usuario.

## Etapas del análisis

* Carga y exploración de los datos.
* Limpieza y preparación de los datasets.
* Unión de las tablas mediante `merge()`.
* Detección y análisis de valores atípicos (outliers).
* Segmentación de clientes por edad y nivel de uso.
* Visualización de resultados con gráficos.
* Elaboración de conclusiones y recomendaciones para el negocio.

## Cómo ejecutar el notebook

1. Descargar el archivo del proyecto (`.ipynb`).
2. Abrirlo en Google Colab o Jupyter Notebook.
3. Ejecutar las celdas en el orden en que aparecen para reproducir el análisis completo.

## Guía de reproducción

Para reproducir este proyecto, asegúrate de contar con los archivos `plans.csv`, `users_latam.csv` y `usage.csv` en la carpeta correspondiente. Luego abre el notebook y ejecuta todas las celdas de forma secuencial para obtener los mismos resultados y visualizaciones.
